# readHelper-chrome-extension
一个打算用来方便人们学习的chrome插件

这个插件目的是方便学习，譬如，将代码或文章粘贴或标记，避免文章上下来回滚动；搜索单词

技术栈： JavaScript + HTML + CSS + Vue.js
