# learn-Helper
一个打算用来方便人们学习的chrome插件

这个插件目的是方便学习与阅读。
当您阅读一篇文章，您可能需要来回滚动获取信息，这会影响您的学习与阅读，您可以通过本插件不断打下标记，来帮助您。详情见下方教程
同时，当您需要搜索时，无需离开当前页面去新建一个搜索页面，而是通过本插件直接搜索，更加方便
本插件的初心是方便学习、方便阅读。

技术栈： JavaScript + HTML + CSS + Vue.js
